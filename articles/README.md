# 文章存储说明

## 当前方案

文章内容目前以内嵌方式存储在以下位置：
- `article.html` - 文章详情页面
- `js/article.js` - 文章内容数据

## 扩展方案

如需扩展为真正的静态文章系统，可以：

1. 将每篇文章保存为独立的 HTML 文件在 `articles/` 目录下
2. 或使用 Markdown 文件 + 客户端渲染
3. 或接入后端 API 动态加载
